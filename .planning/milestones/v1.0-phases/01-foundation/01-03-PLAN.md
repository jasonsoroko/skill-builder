<objective>
Build the conductor state machine that drives pipeline runs end-to-end, and the Click CLI that provides the user-facing entry point.

Purpose: This is the core of Phase 1. The conductor wires together all infrastructure modules (checkpoint, budget, tracing, stubs) into a working pipeline that can transition through all phases, handle feedback loops, persist state, resume, and enforce budget limits. The CLI makes it accessible via `skill-builder build brief.json`.

Output: A working `skill-builder build` command that drives stub agents through the full pipeline, with checkpoint persistence, resume capability, budget enforcement, and dry-run mode.
