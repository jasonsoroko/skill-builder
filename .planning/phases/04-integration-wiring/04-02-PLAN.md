Wire budget recording from agents through the conductor and apply tracing decoration dynamically at dispatch time so every agent call is tracked in LangSmith with phase/agent/iteration metadata.

Purpose: Closes CORE-08 (global token budget cap prevents runaway costs) and OBS-02 (each agent run includes metadata tags for phase, agent name, and iteration) from the v1.0 milestone audit.
Output: All agents return usage metadata, conductor records usage and enforces budget halt, conductor wraps every agent.run() with traceable_agent dynamically.
</objective>
