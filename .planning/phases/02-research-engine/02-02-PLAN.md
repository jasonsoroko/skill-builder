  <behavior>
    Saturation:
    - check_saturation(client, pages, required_capabilities) returns SaturationResult
    - Uses a single cheap Sonnet call via messages.parse(output_format=SaturationResult)
    - Checks if any required capability has zero representation in harvested content
    - Per locked decision: "lightweight pre-filter -- a single short Sonnet call, not a full analysis"
    - max_tokens=1024 (small -- this is a cheap check)

    HarvestAgent:
    - Conforms to BaseAgent Protocol (run(**kwargs) -> HarvestResult)
    - run(brief=SkillBrief, state=PipelineState) orchestrates the full harvest
    - Step 1: Generate search queries via query_generator (LLM with template fallback)
    - Step 2: Build task list -- route_url for each seed URL + exa_search for each Exa query + tavily_search for each Tavily query
    - Step 3: Run all tasks in parallel via asyncio.gather(*tasks, return_exceptions=True)
    - Step 4: Collect pages, log errors (non-fatal), handle github discovered docs URLs (add firecrawl tasks and re-gather)
    - Step 5: Deduplicate via deduplicate()
    - Step 6: Detect versions and check conflicts via version_check
    - Step 7: Run saturation pre-filter
    - Step 8: If saturation says missing capabilities AND this is not a re-harvest, immediately re-harvest with targeted queries (but this is bounded by conductor's gap loop, so just flag in warnings)
    - Step 9: Build and return HarvestResult with pages, warnings, version_conflicts, queries_used
    - Uses asyncio.Semaphore to limit concurrent requests per API (max 3 concurrent Exa, max 3 concurrent Tavily -- per RESEARCH.md Pitfall 3)
  </behavior>
  <action>
    1. Create `src/skill_builder/harvest/saturation.py`:
       - Import SaturationResult from models, create_traced_client from tracing
       - `async def check_saturation(client: Anthropic, pages: list[HarvestPage], required_capabilities: list[str]) -> SaturationResult`
       - Build a concise prompt: list required capabilities and a summary of harvested content (truncate to keep prompt small). Ask: "For each required capability, is there at least some relevant content? List any capabilities with zero representation."
       - Call `client.messages.parse(model="claude-sonnet-4-6", max_tokens=1024, output_format=SaturationResult, messages=[...])`
       - If LLM call fails, return `SaturationResult(is_saturated=True, missing_capabilities=[])` (fail-open -- the Gap Analyzer is the real quality gate)

    2. Create `src/skill_builder/agents/harvest.py`:
       - `class HarvestAgent:` conforming to BaseAgent Protocol
       - Constructor takes `client: Anthropic` (optional, defaults to create_traced_client())
       - `def run(self, **kwargs) -> HarvestResult:` extracts `brief: SkillBrief` and `state: PipelineState` from kwargs
       - Internally calls `asyncio.run(self._harvest(brief, state))` to bridge sync Protocol to async internals
       - `async def _harvest(self, brief, state) -> HarvestResult:`
         a. Generate queries: `generate_search_queries(self.client, brief)` (or `refine_gap_queries` if state indicates re-harvest -- check `state.gap_loop_count > 0` and `state.gap_report` has `recommended_search_queries`)
         b. Build async tasks with semaphores:
            - `url_tasks = [route_url(seed, brief.max_pages) for seed in brief.seed_urls]`
            - `exa_tasks = [sem_exa(exa_search, q) for q in queries.exa_queries]` (rate-limited)
            - `tavily_tasks = [sem_tavily(tavily_search, q) for q in queries.tavily_queries]` (rate-limited)
         c. `results = await asyncio.gather(*all_tasks, return_exceptions=True)`
         d. Collect pages from successful results, log exceptions (non-fatal)
         e. Handle github discovered docs URLs: if any route_url returned a tuple (pages, docs_urls), schedule firecrawl_crawl for each discovered URL and await
         f. `pages = deduplicate(all_pages)`
         g. For each page: `versions = detect_version(page.content); page.detected_version = versions[0] if versions else None`
         h. `conflicts, warnings = check_version_conflicts(pages, brief.target_api_version)`
         i. `saturation = await check_saturation(self.client, pages, brief.required_capabilities)`
         j. If not saturation.is_saturated: add to warnings `f"Saturation: missing content for {', '.join(saturation.missing_capabilities)}"`
         k. Return `HarvestResult(pages=pages, total_pages=len(pages), warnings=warnings, version_conflicts=conflicts, queries_used=queries.exa_queries + queries.tavily_queries)`
